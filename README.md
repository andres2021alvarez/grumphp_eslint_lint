# GrumPHP + ESLint + Stylelint + PHPStan

Este repositorio contiene la configuración base para un proyecto PHP con herramientas de calidad de código automatizadas mediante GrumPHP.

## 🚀 Inicio Rápido

1. Instalar dependencias:
   ```bash
   composer install
   npm install
   ```

## 🛠 Herramientas Incluidas
- **PHP:** PHPStan, PHPMD, PHPCS, PHPCPD.
- **JS/CSS:** ESLint, Stylelint.
- **Automatización:** GrumPHP.

## 📖 Guía de Trabajo
Para que tus contribuciones pasen los controles de calidad, por favor lee nuestra:

👉 **[Guía de Convenciones y Calidad](./GUIA_CONVENCIONES.md)**
